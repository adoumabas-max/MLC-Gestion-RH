# Documentation SMQ — MLC-Gestion-RH (livrable 9)

Référentiel : ISO 9001:2015 · Code du travail camerounais (Loi n° 92/007).
Processus RH = support **S2** dans la cartographie de Mercure Logistics SARL.

## Cartographie des processus
- **Management (M)** : M1 pilotage / revue de direction ; M2 système de management de la qualité.
- **Réalisation (R)** : R1 transit & douane ; R2 transport corridors ; R3 relation client.
- **Support (S)** : S1 finances ; **S2 ressources humaines (outillé par l'application)** ; S3 SI ; S4 achats.
